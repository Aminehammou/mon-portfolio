Framework UI pour ESP32 avec Encodeur Rotatif

Ce projet est une bibliothèque modulaire pour créer des interfaces utilisateur sur écran TFT, contrôlées par un encodeur rotatif. Il est conçu pour les systèmes embarqués basés sur ESP32, et utilise des composants réutilisables comme des boutons, des listes, des graphiques, etc.

Fonctionnalités

  • Gestion de l'encodeur rotatif (clics, rotations).
  • Composants UI modulaires (boutons, listes, graphiques).
  • Thèmes personnalisables.
  • Gestion de l'énergie (mode veille).

Technologies Utilisées

ESP32 C++ TFT_eSPI PlatformIO GitHub
